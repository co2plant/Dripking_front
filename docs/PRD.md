# Dripking Frontend Product Overview

## 문서 목적

이 문서는 Dripking 프론트엔드가 어떤 사용자 경험을 제공하는지 설명하기 위한 제품 소개 문서입니다. 채용 또는 포트폴리오 검토자가 화면 흐름과 구현 의도를 빠르게 이해할 수 있도록 정리합니다.

## 제품 한 줄 소개

Dripking은 술과 여행을 좋아하는 사용자가 술, 양조장, 여행지를 탐색하고 관심 항목을 여행 일정으로 구성할 수 있게 돕는 웹 서비스입니다.

## 프론트엔드 경험 목표

프론트엔드는 Dripking의 핵심 흐름인 "탐색 -> 저장 -> 일정 구성"을 화면에서 자연스럽게 이어주는 역할을 합니다.

- 사용자가 술, 양조장, 여행지 정보를 쉽게 탐색합니다.
- 관심 항목을 위시리스트에 저장합니다.
- 저장한 항목을 여행 일정으로 바로 옮깁니다.
- 지도와 일정 목록을 함께 보며 계획을 정리합니다.
- 리뷰와 인기 신호를 통해 다른 사용자의 반응을 참고합니다.
- 관리자는 콘텐츠와 신고 리뷰, 인기 데이터를 운영합니다.

## 핵심 사용자 흐름

### 1. 콘텐츠 탐색

사용자는 랜딩 페이지에서 검색과 인기 콘텐츠를 확인하고, 술/양조장/여행지 목록으로 이동합니다.

주요 화면:

- `/`
- `/alcoholList`
- `/distilleryList`
- `/destinationList`
- `/alcohol/:id`
- `/distillery/:id`
- `/destination/:id`

### 2. 위시리스트 저장

사용자는 마음에 드는 술, 양조장, 여행지를 위시리스트에 담습니다.

- 비로그인 사용자는 브라우저 로컬 스토리지에 저장합니다.
- 로그인 사용자는 서버 위시리스트와 연동합니다.
- 로그인 후에는 로컬 위시리스트를 서버 저장 흐름으로 연결합니다.

### 3. 여행 일정 구성

사용자는 여행을 만들고, 위시리스트 항목을 세부 일정으로 추가합니다.

- 위시리스트 항목을 버튼 또는 드래그 앤 드롭으로 일정에 추가합니다.
- 일정에는 날짜, 시작 시간, 종료 시간, 주소, 설명을 입력합니다.
- 일정 순서는 Dragula 기반 드래그 앤 드롭으로 조정합니다.
- 지도에서 계획한 장소의 위치를 확인합니다.

### 4. 리뷰와 신고

사용자는 술, 양조장, 여행지 상세 화면에서 리뷰를 확인하고 작성할 수 있습니다.

- 로그인 사용자는 리뷰를 작성, 수정, 삭제할 수 있습니다.
- 사용자는 부적절한 리뷰를 신고할 수 있습니다.
- 관리자는 신고된 리뷰를 검수하고 숨김, 삭제, 해결 처리할 수 있습니다.

### 5. 관리자 운영

관리자는 별도 관리자 라우트에서 서비스 데이터를 운영합니다.

- 운영 요약과 인기 신호 확인
- 사용자 검색, 권한/상태 수정, 삭제
- 카테고리 관리
- 술, 양조장, 여행지 콘텐츠 관리
- 콘텐츠 이미지 업로드
- 신고 리뷰 검수

## 화면 구성

| 화면 | 설명 |
| --- | --- |
| 랜딩 | 검색, 인기 여행지, 인기 술 |
| 술 목록/상세 | 술 카탈로그 탐색, 리뷰, 위시리스트 |
| 양조장 목록/상세 | 생산지 탐색, 지도, 관련 콘텐츠 |
| 여행지 목록/상세 | 여행지 탐색, 지도, 리뷰 |
| 여행 일정 편집 | 위시리스트 기반 일정 생성과 순서 편집 |
| 사용자 정보 | 프로필과 비밀번호 관리 |
| 관리자 대시보드 | 운영 현황, 인기 여행지/술, 최근 활동 |
| 관리자 콘텐츠 관리 | 술, 양조장, 여행지, 카테고리 관리 |
| 관리자 리뷰 검수 | 신고 리뷰 검토와 조치 |

## 주요 구현 포인트

| 영역 | 구현 의도 |
| --- | --- |
| Pinia 상태 관리 | 인증, 위시리스트, 여행, 일정 데이터를 화면 간 공유합니다. |
| 로컬 우선 저장 | 비로그인 사용자도 탐색과 일정 구성을 먼저 시작할 수 있습니다. |
| 서버 동기화 | 로그인 후 로컬 데이터를 서버 저장 흐름으로 연결해 데이터 손실을 줄입니다. |
| 공통 API 서비스 | `VUE_APP_API_URL`, 토큰 헤더, HTTP 메서드, JSON 파싱을 한곳에서 관리합니다. |
| 권한 라우팅 | 사용자 전용 화면과 관리자 화면을 Vue Router guard로 보호합니다. |
| 지도 연동 | Google Maps 기반으로 목록/상세/일정의 위치 정보를 표시합니다. |
| 인기 콘텐츠 | 7일, 30일, 전체 기간별 인기 여행지와 인기 술을 화면에 보여줍니다. |
| 관리자 UI | 운영자가 콘텐츠 품질과 신고 리뷰를 관리할 수 있는 화면을 제공합니다. |
